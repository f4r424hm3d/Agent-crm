## AdminCreateAgent Form Redesign

### Overview
Redesigned the `AdminCreateAgent.jsx` form to match the UI/UX and functionality of the Agent Registration form (`PartnerApplicationForm.jsx`).

### Key Changes
- **Form Structure**: Transformed from 5 steps to 6 steps (added Documents step).
- **Styling**: Updated from Indigo theme to Emerald theme to match agent-facing pages.
- **Fields Added**: Added 10+ new fields including `alternatePhone`, `qualification`, `city`, `state`, `pincode`, `registrationNumber`, `servicesOffered`, `currentStudents`, `teamSize`, `annualRevenue`, `marketingBudget`, `references`, and `additionalInfo`.
- **Document Uploads**: Added a new Documents step (Step 5) with optional upload functionality for 7 document types.
- **OTP Verification**: Removed OTP verification step (Admin enters email directly).
- **Validation**: Implemented validation for all new fields and file uploads.
